# Code of Conduct

## Our Standards

- Be respectful and constructive in all project interactions.
- Focus reviews on technical quality, not people.
- Assume good intent and ask clarifying questions before escalating.
- Do not harass, insult, or discriminate against others.

## Scope

This code of conduct applies to:

- project repositories and pull requests
- issue discussions
- public and private communication channels related to this project

## Reporting

Report unacceptable behavior by opening a private security report as described in `SECURITY.md`.

## Enforcement

Maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.

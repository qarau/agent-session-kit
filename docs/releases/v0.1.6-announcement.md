# Agent Session Kit v0.1.6 Announcement Copy

## Short Post

Agent Session Kit `v0.1.6` is live.

This release adds a soft task-flow loop for AI-assisted development:

- `nextTask.mjs` recommends what to do next from `docs/session/tasks.md`
- `completeTask.mjs` closes the current task and promotes the next one
- a non-blocking `post-commit` reminder keeps momentum after each commit

Result: fewer stalled sessions, clearer handoffs, and less “what’s next?” overhead.

Release notes: `docs/releases/v0.1.6.md`

## Blog Intro

`v0.1.6` focuses on workflow continuity. ASK now includes a built-in, non-blocking reminder loop that surfaces the next task after commits and keeps session taskboards moving forward automatically. Teams keep strict guardrails where they matter (branch/worktree and session freshness checks) while gaining a lighter daily flow for planning and execution.

# Agent Session Kit v0.1.6

Release date: 2026-03-11

## Summary

This release adds a soft task-flow reminder loop so agent/developer sessions can move forward without repeatedly asking what to do next.

## Highlights

- Added `scripts/session/nextTask.mjs` to print the recommended next task and a short queue from `docs/session/tasks.md`.
- Added `scripts/session/completeTask.mjs` to:
  - mark the current task complete
  - promote the next task into `Now` when needed
  - stamp `Last updated`
  - print the next recommendation
- Added `.githooks/post-commit` soft reminder that runs `node scripts/session/nextTask.mjs` (non-blocking).
- Updated installer/hook wiring so `post-commit` is included and executable on POSIX targets.
- Added smoke-test coverage for install and behavior of `nextTask.mjs` and `completeTask.mjs`.

## Verification

- `npm run test` (from `agent-session-kit/`) passed.

## Upgrade Notes

- Existing ASK adopters should re-run:
  - `node scripts/session/installHooks.mjs`
- No breaking contract changes in existing pre-commit/pre-push enforcement.

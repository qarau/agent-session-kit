# Agent Session Kit v3.0.0 Announcement Copy

## Short Post

Agent Session Kit `v3.0.0` is live.

ASK 3.0 is the Session OS release: runtime governance now spans planning, execution, coordination, and delivery with an event-ledger core.

Highlights:

- event-ledger runtime with replayable projection snapshots
- full runtime command lanes for session, execution, and delivery governance
- multi-agent routing/claims/child-session coordination
- guarded execution reliability (`180s` timeout + auto-retry) and runtime diagnostics

Result: teams get policy-safe, replayable, and more autonomous developer-agent execution from first edit to push.

Release notes: `docs/releases/v3.0.0.md`

## Blog Intro

`v3.0.0` is a major ASK release that upgrades the runtime from commit/push gate enforcement into a Session OS model. ASK now records deterministic event history, replays it into runtime snapshots, and exposes contract-first commands across session lifecycle, integration readiness, multi-agent coordination, and delivery governance. The outcome is a platform-agnostic runtime that keeps teams shippable through long-running agent work without depending on brittle script-path behavior.
